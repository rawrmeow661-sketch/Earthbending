# Plane Crazy Earthbending Script

This is a Roblox script for the game Plane Crazy that allows you to perform "earthbending" by placing a sphere of blocks around your character.

## Features

- GUI with options for radius and block type
- Places blocks in a spherical formation around the player

## How to Use

1. Copy the loadstring below.
2. Paste it into your Roblox executor (e.g., Synapse X, Krnl).
3. Execute the script in Plane Crazy.
4. Use the GUI to set options and activate.

## Loadstring

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/rawrmeow661-sketch/Earthbending/main/earthbending.lua"))()
```

Replace `yourusername` with your actual GitHub username.

## Installation

1. Clone or download this repository.
2. Upload `earthbending.lua` to your GitHub repository.
3. Update the loadstring URL accordingly.
